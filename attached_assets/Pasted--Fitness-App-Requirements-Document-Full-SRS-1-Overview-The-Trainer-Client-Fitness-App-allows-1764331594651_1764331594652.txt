8. Non-Functional Requirements (NFRs)

NFR-2 Real-time chat must deliver messages within 1 second.

8.2 Security

NFR-3 All personal data is encrypted in the database.
NFR-4 Passwords stored with hashing (bcrypt/argon2).
NFR-5 Role-based permissions enforced for all API calls.

8.3 Availability

NFR-6 System uptime target: 99%.
